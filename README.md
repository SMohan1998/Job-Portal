# Job Portal (MERN Stack)

A full-stack Job Portal application built with the **MERN Stack** enabling smooth interaction between **Job Seekers** and **Employers**.  
The platform provides **job postings, job applications, resume uploads, job recommendations, company profiles, and an applicant tracking system (ATS)**.

---

## Features

### User Authentication & Role Management
- Registration & Login for **Job Seekers** and **Employers**
- JWT-based secure authentication
- Role-based access control
- Password reset & update functionality

### Job Postings (Employer)
- Create, edit, delete job posts
- Post includes: title, description, requirements, salary, and location
- View all jobs posted by employer

### Job Applications (Job Seeker)
- Apply to jobs using uploaded resume
- Track application status: **Applied, Interviewed, Rejected, Selected**
- Employers can track applications (ATS)

### Resume Upload & View
- Upload resume (**PDF, DOC, DOCX**)
- Manage profile information & resume files

### Job Recommendation System
- Suggests jobs based on:
  - Skills
  - Past Applications
  - User Preferences
- Automatically filters applied jobs

### Company Profiles
- Create / manage company profile
- Company logo upload
- View job listings posted by the company

### Job Seeker Profile
- Add experience, education, skills, contact details
- Profile picture upload

### Employer ATS
- View applicants for jobs
- Track statuses & manage candidate pipeline
